e:Tc2e,<h1>Магия Tailwind CSS</h1>
<p>Tailwind CSS - это утилитарный CSS-фреймворк, который в корне меняет подход к стилизации веб-приложений. Вместо написания собственных CSS-классов, мы используем предопределенные утилитарные классы для создания дизайна.</p>
<h2>Почему Tailwind?</h2>
<ol>
<li>
<p><strong>Производительность разработки</strong></p>
<ul>
<li>Нет необходимости придумывать имена классов</li>
<li>Мгновенный результат без переключения между файлами</li>
<li>Легко экспериментировать с дизайном</li>
</ul>
</li>
<li>
<p><strong>Оптимизация</strong></p>
<ul>
<li>Автоматическое удаление неиспользуемых стилей</li>
<li>Минимальный размер финального CSS</li>
<li>Отличная производительность</li>
</ul>
</li>
<li>
<p><strong>Гибкость</strong></p>
<ul>
<li>Полная кастомизация через конфигурацию</li>
<li>Создание собственных утилит</li>
<li>Интеграция с любым JavaScript-фреймворком</li>
</ul>
</li>
</ol>
<h2>Основные концепции</h2>
<h3>Утилитарные классы</h3>
<pre><code class="language-html">&#x3C;div class="p-4 bg-white rounded-lg shadow-md hover:shadow-lg transition-shadow">
  &#x3C;h2 class="text-xl font-bold text-gray-800 mb-2">Заголовок&#x3C;/h2>
  &#x3C;p class="text-gray-600">Описание&#x3C;/p>
&#x3C;/div>
</code></pre>
<h3>Отзывчивый дизайн</h3>
<p>Tailwind использует модификаторы для создания отзывчивого дизайна:</p>
<ul>
<li><code>sm:</code> - от 640px</li>
<li><code>md:</code> - от 768px</li>
<li><code>lg:</code> - от 1024px</li>
<li><code>xl:</code> - от 1280px</li>
<li><code>2xl:</code> - от 1536px</li>
</ul>
<pre><code class="language-html">&#x3C;div class="text-sm md:text-base lg:text-lg">
  Адаптивный текст
&#x3C;/div>
</code></pre>
<h3>Темная тема</h3>
<p>Tailwind делает реализацию темной темы простой:</p>
<pre><code class="language-html">&#x3C;div class="bg-white dark:bg-gray-800 text-gray-900 dark:text-gray-100">
  Контент с поддержкой темной темы
&#x3C;/div>
</code></pre>
<h2>Заключение</h2>
<p>Tailwind CSS предоставляет мощный и гибкий инструментарий для создания современных веб-интерфейсов. Его подход к стилизации может показаться необычным поначалу, но после небольшой практики вы оцените все преимущества этого фреймворка.</p>
